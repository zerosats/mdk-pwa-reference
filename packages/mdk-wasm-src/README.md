# MDK-WASM Source

This is the WASM binding crate for MDK. It wraps the MDK Rust library for use in JavaScript/TypeScript.

## Prerequisites

1. **Rust 1.90+**
2. **LLVM** (for secp256k1 WASM compilation):
   ```bash
   brew install llvm
   ```
3. **wasm-pack**:
   ```bash
   cargo install wasm-pack
   ```
4. **WASM target**:
   ```bash
   rustup target add wasm32-unknown-unknown
   ```

## Building

### Option 1: Using wasm-pack (recommended)

```bash
# Set LLVM environment
export LLVM_PATH="/opt/homebrew/opt/llvm"
export CC_wasm32_unknown_unknown="${LLVM_PATH}/bin/clang"
export AR_wasm32_unknown_unknown="${LLVM_PATH}/bin/llvm-ar"

# Build for bundlers (webpack/vite)
wasm-pack build --target bundler --release

# Or for web (direct script tag usage)
wasm-pack build --target web --release
```

### Option 2: Manual cargo build

```bash
CC_wasm32_unknown_unknown="/opt/homebrew/opt/llvm/bin/clang" \
AR_wasm32_unknown_unknown="/opt/homebrew/opt/llvm/bin/llvm-ar" \
cargo build --target wasm32-unknown-unknown --release
```

## Important: MDK Workspace Modification

This crate depends on MDK. Before building, the MDK workspace `Cargo.toml` must have OpenMLS configured with the `js` feature:

```toml
openmls = { version = "0.8", default-features = false, features = ["js"] }
```

Without this, WASM compilation will fail with:
```
error: In order for OpenMLS to build for WebAssembly, JavaScript APIs must be available...
```

## API

### Initialization
```javascript
import init, { mdk_init } from 'mdk-wasm';

await init();  // Load WASM module
mdk_init();    // Initialize MDK
```

### Key Package (for joining groups)
```javascript
const keyPackage = create_key_package(
  "npub1...",                    // Your nostr pubkey
  '["wss://relay1.com"]'        // Relay URLs JSON
);
console.log(keyPackage.event_json);
```

### Create Group
```javascript
const result = create_group(
  "npub1...",                    // Creator pubkey
  '[...]',                       // Key package events JSON
  JSON.stringify({
    name: "My Group",
    description: "A test group",
    admins: ["npub1..."],
    relays: ["wss://relay1.com"]
  })
);
console.log(result.group_id);
console.log(result.welcome_event_jsons);  // Send to invitees
```

### Send Message
```javascript
const msg = create_message(
  "abc123...",                   // Group ID (hex)
  "Hello world!",                // Message content
  "npub1..."                     // Sender pubkey
);
// Publish msg.event_json to Nostr
```

### Process Received Message
```javascript
const result = process_message(event_json);
if (result.message_type === "application") {
  console.log(`From ${result.sender}: ${result.content}`);
}
```
